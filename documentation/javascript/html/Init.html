class Init
extends Base


The Init class is supposed to be used on load of page.
Right now the function load() is used as an argument for the thirdparty lib method Behaviour.addLoadEvent(Init.load); and is not supposed to called manually. If you you want something to happen on load of page, add another prototype function (e.g. getVars()) to class Init and call it within the load() function instead.

Example (Schema): 


function Init_load ()
{	
	try {
		// Do something while page load
		Init.getVars();
		...
		}
	} catch (e) {
		_applyError(e);
	}
}
