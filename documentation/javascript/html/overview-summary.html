WelcomposeTM Javascript API

This document is the API Specification for WelcomposeTM Javascript API.

Summary

The Welcompose is a content management system based on a PHP5/mysql enviroment. For detailed expanation please have a look at the application docs.

The welcompose javascript enviroment uses the abilities of some thirdparty libs mentioned below. These are not documented in this core documentation.

Used thirdparty libs are:
Prototype for Ajax calls and customized functions (http://prototype.conio.net/)
Behaviour to use css selectors avoiding inline event handlers (http://bennolan.com/behaviour/)
Scriptaculous for visual effects only (http://script.aculo.us/)
